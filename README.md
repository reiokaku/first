# 日报生成助手

主文件是 [index.html](/W:/reiok/Documents/日报/index.html)。

现在这个版本已经整理成可部署的静态网页，适合放到网址上给手机访问。

## 已经补好的能力

- 手机优先布局
- 可在线部署为网址
- 支持加入手机桌面
- 支持离线缓存，打开过一次后网络差时也更稳
- 名单保存在浏览器本地
- 学生和班级都支持线上/线下

## 最简单的部署方式

### 方法一：Netlify Drop

最适合不会写代码部署流程的时候用。

1. 打开 Netlify Drop 网站。
2. 把这个文件夹里的这些文件一起拖进去：
   - [index.html](/W:/reiok/Documents/日报/index.html)
   - [manifest.webmanifest](/W:/reiok/Documents/日报/manifest.webmanifest)
   - [sw.js](/W:/reiok/Documents/日报/sw.js)
   - [icon-192.svg](/W:/reiok/Documents/日报/icon-192.svg)
   - [icon-512.svg](/W:/reiok/Documents/日报/icon-512.svg)
3. 它会自动给你一个网址。
4. 用手机打开这个网址即可使用。

### 方法二：GitHub Pages

适合以后长期维护版本。

1. 新建一个 GitHub 仓库。
2. 把上面的文件上传到仓库根目录。
3. 打开仓库设置里的 Pages。
4. 选择从主分支发布。
5. 几分钟后会生成一个固定网址。

### 方法三：Vercel

也很适合静态网页，部署过程和 Netlify 类似。

## 手机上怎么用

1. 用手机浏览器打开部署后的网址。
2. 选择班级或学生。
3. 调整日期、时间、时长和内容。
4. 点击“复制日报”。
5. 如果浏览器支持，可以选择“添加到主屏幕”。
